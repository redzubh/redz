## 🛠️ Troubleshooting Common Issues

*   **Engine Failed to Load (Error 0x05)**
    *   *Description*: The execution manager is unable to attach itself to the target process.
    *   *Solution*: **Ensure you are running the application as an administrator and that your antivirus has the app folder marked as an exclusion.**

*   **UI Does Not Appear in Game**
    *   *Description*: The main hub dashboard is working in the background but fails to overlay on screen.
    *   *Solution*: **Try cycling the visibility hotkey (Insert/F8) or re-injecting the application.**

*   **Script Load Fail (HTTP Error 404)**
    *   *Description*: The launcher cannot fetch files from the remote server.
    *   *Solution*: **Check your local network connection or disable active VPN software before launching.**

*   **Out of Memory Crash**
    *   *Description*: The game client crashes due to memory limitations during heavy execution.
    *   *Solution*: **Clear background applications and reduce your config-defined script logging settings.**
